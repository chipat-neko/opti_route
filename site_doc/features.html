01 — Calcul itineraire

Optimisation VROOM

Calcul d'ordre optimal des arrets via OpenRouteService (algorithme VROOM open-source). 500 optimisations gratuites par jour sur le quota dev free.

  • Profil driving-car (VL) ou driving-hgv (camion lourd, restrictions PL)
  • Contrainte capacite vehicule (refuse les arrets qui depassent)
  • Fenetres horaires obligatoires ou preferees
  • Priorites EN 1ER, EN DERNIER, A EVITER
  • Evitement peages (toggle)
  • Trace GeoJSON polyline affichee sur la carte
  • Calcul ETA prorata + chrono live avec pause/reprise
  • Fallback haversine si /directions echoue (pas de plantage)
02 — Saisie rapide

Scan bordereaux OCR

OCR on-device via Google ML Kit (script latin). Pas de cloud, pas d'upload. Auto-detection du format de bordereau pour appliquer le bon parser.

  • Parser MESEXP (format Noah, ~85% taux carte verte)
  • Parser Colissimo / La Poste (tracking 6A/6L/8L...)
  • Parser Chronopost (tracking XR/XE...FR)
  • Auto-detection format puis fallback parser generique
  • Extraction : nom destinataire, rue, CP, ville, tel, nb colis
  • Confiance high / low / none affichee dans une carte coloree
  • Indice de confiance pour decider de pre-remplir ou demander confirmation
03 — Multi-personnes (Phase 1 local)

Mode chef d'equipe

Active dans Parametres → Entreprise & equipe. 100% local au telephone du chef. Pour la vraie sync multi-appareils (chaque coequipier sur son propre phone), roadmap Phase 2 CB avec backend Firebase ou Supabase.

  • Mon equipe : CRUD coequipiers (nom, telephone, couleur avatar, archivage non-destructif)
  • Affecter un arret a un coequipier depuis la bottom sheet
  • Affecter tous les nouveaux arrets d'une tournee a Lucas (champ "par defaut")
  • Bulk : "Affecter le reste a..." (tous les arrets non affectes)
  • Filtre par coequipier dans la liste d'arrets (chips couleur)
  • Tableau de bord agrege : breakdown par personne, badge EN PAUSE temps reel
  • Partage WhatsApp cible : ouvre directement la conv de Lucas avec ses arrets
  • PDF individuel par coequipier (FICHE DE Lucas + filtres stops + footer SIRET)
  • Stats par coequipier (livres / echecs / colis / taux%)
04 — Memoire metier

Carnet d'adresses local enrichi

Auto-alimentation a chaque arret valide. Cherche par nom, ville, adresse, tag. Filtre par couleur, favori, ou tag custom.

  • Tags multiples (separes par virgules)
  • Code interphone / portail (champ dedie)
  • Batiment / etage / appartement
  • Photo facade (path local prive)
  • Notes pre-definies (pre-remplies au prochain arret)
  • Couleur avatar (lime/emerald/amber/red/cream/ink)
  • Etoile favori (remonte en haut)
  • Stats par client : nb livraisons, taux reussite, top raisons d'echec
  • Import / export CSV (RFC 4180) + vCard 3.0 (compatibles Contacts Android)
05 — Mode terrain

Suivi temps reel de tournee

  • Chrono "Demarree il y a Xh Ymin" en haut
  • Pause / Reprise courte (pause dejeuner) avec cumul des secondes pausees
  • Badge EN PAUSE amber visible dans le bandeau
  • Badge ETA "≈14:32" sur chaque arret a livrer
  • Prochain arret avec distance vol d'oiseau
  • Tri par distance GPS au lieu de l'ordre optimise
  • Recherche par nom / rue / notes
  • Filtre par statut (a livrer / livres / echecs)
  • Drag-and-drop pour reorganiser manuellement
  • Photo preuve (avant ou apres validation)
  • Annuler le dernier statut (1 tap, fenetre de grace)
  • Position GPS enregistree au moment du "Marquer livre / echec" (preuve passage)
  • Notification post-tournee : recap livres / echecs / duree
06 — Donnees & analyses

Statistiques cumulatives

  • 3 fenetres : 7 jours / 30 jours / 365 jours
  • Carte motivation lime : colis livres annee + km + tournees + streak
  • Badge taux reussite annuel coloré (≥95% emerald, ≥85% lime, sinon amber)
  • Top 5 clients (avec couleur avatar du carnet)
  • Top 5 raisons d'echec globales (barres relatives)
  • Heures travaillees par jour de la semaine
  • Colis livres par jour de la semaine (carte 7 jours)
  • Stats par coequipier (mode chef)
  • Cout carburant cumule (params EUR/L + L/100km configurables)
  • Comparatif M-1 / M-12 (evolution)
  • Export CSV au standard RFC 4180 (compatible Excel / Sheets / dashboard web)
07 — Sortie de donnees

Exports & partage

  • PDF recap tournee (avec footer SIRET + slogan entreprise)
  • PDF par coequipier (pill "FICHE DE Lucas" en header)
  • Texte court pour WhatsApp / SMS / mail (formate liste numerotee)
  • Partage cible coequipier via lien wa.me (numero pre-rempli)
  • CSV stats (365 jours, formate Excel-ready)
  • CSV carnet (sauvegarde / migration)
  • vCard 3.0 du carnet (import direct dans Contacts Android)
  • Tableau de bord web (uploader le CSV pour visualiser sur grand ecran)
08 — Personnalisation

Themes & accessibilite

  • 4 palettes : Lime (defaut) / Ocean (bleus) / Terracotta (terre) / Mono (NB)
  • Mode clair / sombre / auto (suit le systeme)
  • Mode XL : polices x1.15 pour la conduite
  • Contraste eleve (lecture en plein soleil)
  • Toggle rappel veille auto (notif "prepare tes colis" la veille a 21h)
  • App de navigation par defaut : Google Maps ou Waze
  • Cle ORS configurable (gratuit 500/jour)
09 — Donnees

Securite & vie privee

  • Toutes les donnees restent sur le telephone (SQLite via Drift)
  • Aucun serveur opti_route (sauf APIs publiques BAN / SIRENE / Photon / ORS)
  • Photos preuves dans le repertoire prive de l'app (jamais expose)
  • Pas de tracking analytique, pas de cookies, pas de pub
  • Code source ouvert (audit public possible)
  • Permissions Android minimales (camera, GPS, notifs, internet ORS uniquement)
  • Verrouillage PIN + biometrie (empreinte / face) au demarrage, optionnel
  • Sauvegarde complete (.zip portable avec DB + photos preuves), restore par swap differe au boot
  • Auto-backup local periodique (hebdo / mensuel) avec rotation des 5 derniers
  • Export CSV + vCard du carnet d'adresses pour migration ou archivage
