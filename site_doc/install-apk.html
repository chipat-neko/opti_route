Installation

Installer l'APK sur Android.

opti_route n'est pas encore sur le Play Store (Phase 2). Pour l'instant, tu installes l'APK directement (sideload). C'est rapide, sans inscription, et tu gardes la main sur les mises a jour.

Securite Android. Android te previendra que l'app vient d'une "source inconnue". C'est normal : ce n'est pas le Play Store. Le code source est public sur GitHub si tu veux verifier ce qui tourne sur ton phone.

Quelle variante telecharger ?

Recommande
arm64-v8a
~39 MB · Tous les phones modernes (>2017)
Vieux phones
armeabi-v7a
~12 MB · Phones 2014-2016
Universel
universal
~57 MB · Marche partout, plus lourd

En cas de doute, prends arm64-v8a. Si l'install echoue avec "le package n'est pas compatible", essaie universal.

Procedure

1

Telecharger le fichier APK

Va sur la page des releases GitHub depuis ton telephone. Sous la derniere release, telecharge le fichier app-arm64-v8a-release.apk (ou la variante adaptee).

Si ton navigateur te demande "Telecharger ce type de fichier ?", confirme.

2

Autoriser les sources inconnues (1x suffisant)

Sur Android 8+, l'autorisation se donne par application (Chrome, Drive, Files, etc.). Quand tu ouvres l'APK telecharge, Android propose un bouton Parametres. Clique dessus puis active Autoriser depuis cette source pour le navigateur ou l'app fichiers que tu utilises.

Sur les Android plus anciens : Parametres → Securite → coche Sources inconnues.

3

Ouvrir l'APK et confirmer l'installation

Reviens sur la notification de telechargement ou ouvre l'app Fichiers → Telechargements. Tape sur app-arm64-v8a-release.apk.

Android affiche une fenetre "Installer cette application ?". Confirme avec Installer. Quelques secondes apres, l'app opti_route apparait dans ton tiroir d'applications.

4

Premier lancement & cle OpenRouteService

L'app demarre sur l'onboarding qui explique comment recuperer ta cle gratuite OpenRouteService (500 optimisations/jour). openrouteservice.org/dev → cree un compte gratuit → copie la cle → colle-la dans Parametres.

Sans cle, tu peux quand meme utiliser le mode manuel : ajouter des arrets sans optimisation et naviguer via Maps/Waze a cote.

Mettre a jour l'app

Pas de mise a jour auto en Phase 1. Quand une nouvelle version sort, tu telecharges le nouvel APK et tu re-installes par-dessus. Tes donnees (tournees, carnet, photos) sont conservees. Android remplace juste le code de l'app.

Active les notifications GitHub sur le repo chipat-neko/opti_route pour etre alerte des nouvelles releases. Tu peux aussi suivre le changelog.

Probleme courant

"Le package n'est pas compatible"

Ton phone n'a pas la bonne architecture CPU. Telecharge la variante universal (un peu plus lourde mais marche partout).

"App non installee" sans autre detail

Probablement un APK corrompu pendant le download. Relance le telechargement (clic droit ou re-clic, "Telecharger a nouveau") et re-tente.

"Probleme d'analyse du package"

Memoire pleine ou Android trop ancien (<5.0 Lollipop). Verifie l'espace dispo (Parametres → Stockage) et la version Android (Parametres → A propos du telephone).
