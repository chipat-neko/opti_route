Pourquoi opti_route

Toutes les fonctions d'un GPS de livraison pro, gratuit.

Construit avec un livreur reel pour les besoins reels. Pas de feature commerciale gadget.

Optimisation VROOM

Itineraire optimal calcule via OpenRouteService (500/jour gratuit). Profil voiture ou camion, contraintes capacite et fenetres horaires.

Scan bordereaux

OCR on-device (Google ML Kit). Detection auto MESEXP, Colissimo, Chronopost. Extraction nom + adresse + CP + ville + telephone.

Gestion equipe

Affecte tes arrets a tes coequipiers. Vue agregee pour le chef. Partage WhatsApp/SMS cible par personne. PDF individuel par coequipier.

Suivi temps reel

ETA par arret, chrono avec pause/reprise, badge "EN COURS" sur le drawer, notifications post-tournee, recap automatique.

Photo preuve

Capture photo signature ou colis depose a la validation. Stockage local prive. Sert en cas de litige client.

4 palettes & mode sombre

Lime, Ocean, Terracotta, Mono. Mode XL pour la conduite, contraste eleve pour le plein soleil. Adapte a chaque chauffeur.

Carnet local enrichi

Tags multiples, codes interphone, etage / batiment, photo facade, historique livraisons par client. Import/export CSV + vCard.

100% prive

Toutes les donnees restent sur ton telephone. Aucun serveur, aucune CB requise. Le SIRET et le RGPD t'appartiennent.

Export PDF / CSV

PDF recap avec ton SIRET, slogan, footer entreprise. CSV stats pour Excel. Partage natif vers email, Drive, etc.

Mode chef d'equipe

Reparti tes tournees, suis l'avancement, partage les fiches.

Active le toggle "Mode chef" dans Parametres. Tu gagnes un tableau de bord agrege, l'affectation en masse et l'export PDF par personne.

Tableau de bord

Vue agregee toutes les tournees du jour : avancement par personne, total arrets, livres / restants, badge "EN PAUSE" temps reel.

Affectation en masse

"Affecter le reste a Lucas" en un tap. Tournee : champ "Affecter par defaut" qui pre-rempli tous les nouveaux arrets crees.

PDF individuel

Un PDF par coequipier, avec "FICHE DE Lucas" en pill lime, ses arrets uniquement, ton SIRET en footer. Partage groupe en un tap.

Partage cible WhatsApp

Bouton "Partager a Lucas" qui ouvre WhatsApp pre-rempli avec ses arrets, son numero pre-selectionne. Fallback SMS auto.

Suivi chef d'equipe sur le web

Exporte ton CSV depuis Statistiques sur ton phone, puis ouvre le tableau de bord web pour voir tes tournees en grand, partager des resumes, faire ta compta.

Comment ca marche

4 etapes pour commencer en moins de 5 minutes.

1

Installe l'APK

Telecharge l'app sur ton Android. Pas de Play Store requis en phase 1 (gratuit / sandboxing prive).

2

Configure ORS

Cle API OpenRouteService gratuite (500 optimisations/jour). 30 secondes sur openrouteservice.org/dev.

3

Cree ta tournee

Scan bordereaux ou tape les adresses. L'app optimise l'ordre, calcule le trajet, propose Maps/Waze.

4

Suis ton equipe

Active "Mode chef d'equipe" dans Parametres. Ajoute tes aidants. Affecte. Partage. Exporte sur le web.
