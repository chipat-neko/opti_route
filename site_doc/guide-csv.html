Exporter ton CSV pour le tableau de bord.

Le tableau de bord web lit un fichier CSV genere par l'app. En 3 etapes tu auras tes tournees visualisees sur grand ecran.

1

Ouvre Statistiques dans l'app

Drawer (menu en haut a gauche) → Statistiques. Tu y vois le recap 7j / 30j / 365j, la carte de motivation lime et le breakdown par coequipier.

2

Tape "Exporter en CSV"

Tout en bas de l'ecran Stats, la carte "EXPORTER LES STATS" a un bouton bleu/lime "Exporter en CSV". Tap. Le selecteur natif Android propose ou envoyer le fichier :

  • Gmail / Outlook (te l'envoyer a toi-meme)
  • Google Drive (sync automatique avec ton PC)
  • Telegram / WhatsApp (te l'envoyer)
  • Bluetooth direct vers ton PC
3

Charge-le dans le dashboard web

Ouvre le tableau de bord sur ton ordinateur. Glisse-depose ton fichier .csv sur la zone de drop, ou clique pour parcourir. Le viewer affiche immediatement :

  • 5 stat cards (Tournees / Arrets / Colis / Km / Duree)
  • 3 graphes (colis livres, statuts, distance)
  • Tableau brut trie
  • Boutons re-export propre + impression PDF

Format du CSV (pour les curieux)

Le fichier respecte le format RFC 4180 (compatible Excel, Google Sheets, Numbers). Le header est : 

date,nom,statut,arrets,colis_livres,distance_km,duree_min,pause_min
2026-05-13,Tournee mardi,terminee,15,22,52.1,210,20

Le statut peut etre brouillon, optimisee, en_cours ou terminee. La duree exclut les pauses (cumul du chrono effectif).

100% local au navigateur. Aucun upload n'est fait vers un serveur opti_route ou un tiers. Le CSV reste sur ton ordinateur, parse en memoire.
