Roadmap opti_route.

Le projet est decoupe en 3 phases, du gratuit-local au pro-flotte. Chacune ajoute des features sans casser les precedentes. Les phases sont independantes : tu peux rester en Phase 1 indefiniment sans aucun cout.

1

Phase 1 — Local & gratuit

Avril 2026 → en cours · Aucune CB · Tout reste sur le telephone
LIVE

Tout ce qu'un livreur ou un petit chef d'equipe local peut vouloir. Aucun backend, aucun abonnement. La base SQLite Drift reste sur le telephone. Les APIs externes utilisees (BAN, SIRENE, Photon, OpenRouteService) sont publiques et gratuites, avec des quotas confortables.

Optimisation VROOM (500/jour gratuit)
Scan bordereaux OCR (3 formats)
Gestion equipe locale (un seul phone)
Carnet enrichi (tags, codes, photos)
Stats annuelles + top raisons echec
Exports PDF, CSV, vCard, texte
4 palettes + mode XL + contraste
Dashboard web (lecture CSV)
2

Phase 2 — Multi-appareils & cloud

Quand un budget ~25 USD/mois est valide · Backend Firebase ou Supabase
PROCHAIN

Le chef d'equipe gere son equipe a distance. Chaque coequipier a son propre telephone avec son propre compte. Les tournees sont envoyees du chef vers les livreurs en temps reel, sans CSV manuel. Les statuts remontent automatiquement.

Auth (email + password ou OAuth Google)
Sync temps reel (Firestore listeners)
Dashboard chef live (positions GPS coequipiers)
Notification push "X a fini sa tournee"
Backup cloud automatique chiffre
Publication Play Store (25 USD une fois)
Web app temps reel (suivi distant)
ORS premium (10K+ optimisations/mois)
3

Phase 3 — Pro & flotte

Si l'app trouve sa cible · Plus de budget API + dev time
PLUS TARD

Outils pour entreprises de livraison structurees : plusieurs vehicules, RH simple, integration ERP, GPS turn-by-turn embarque.

GPS turn-by-turn integre (Mapbox SDK)
Optimisation multi-vehicules (VROOM avance)
Android Auto / CarPlay
Facturation auto (PDF tournee -> client)
Integration API client (webhooks)
Suivi temps de conduite (RGPD compliant)
Engagement Phase 1 : tant que les phases 2 et 3 ne sont pas livrees, la Phase 1 reste gratuite et fonctionnelle. Aucun "dark pattern" pour pousser au CB.
