Ecran de verrouillage.

Affiche au demarrage de l'app si le verrou est active. Tape sur les chiffres pour saisir le PIN (1234 dans la demo). L'icone empreinte en bas declenche la biometrie systeme.

Saisis ton code

Comment ca marche

  • PIN 4 chiffres stocke en SHA-256 dans la table parametres. Le PIN en clair n'est jamais persiste.
  • Biometrie via le package local_auth Flutter qui delegue au TEE Android (Keystore + StrongBox / TrustZone). Aucune empreinte ne transite par l'app.
  • Auto-lock apres N minutes (5 par defaut) en arriere-plan. Configurable dans Parametres -> Securite.
  • Au cold start : LockScreen s'affiche systematiquement si verrou actif. Pas de skip possible.
