## Description longue (max 4000 caractères)

```
opti_route est l'app que j'ai conçue pour les chauffeurs-livreurs qui en ont marre
des solutions à 50€/mois ou des plannings papier griffonnés au feutre.

L'app calcule l'ordre optimal de tes arrêts pour minimiser le temps et les
kilomètres, gère ton carnet client local, et scanne automatiquement les
bordereaux de livraison via l'appareil photo. Tout reste sur ton téléphone :
pas de serveur, pas d'abonnement, pas de carte bancaire à sortir.

== POUR QUI ==

• Chauffeurs-livreurs indépendants
• Auto-entrepreneurs en livraison à domicile
• Coursiers urbains, livreurs ruraux
• Toute personne qui doit enchaîner 5 à 50 arrêts par jour

== FONCTIONNALITÉS ==

OPTIMISATION DE TOURNÉE
• Calcul de l'ordre optimal via OpenRouteService (plan gratuit 500 tournées/jour)
• Réordonnancement manuel par drag-and-drop pour ajuster les sens uniques
• Arrêts "obligatoire en 1er" ou "en dernier" pour gérer les contraintes terrain
• Fenêtres horaires par arrêt (le matin / l'après-midi)
• Capacité max du véhicule (alerte si dépassement)

CARNET D'ADRESSES LOCAL
• Auto-complétion des adresses déjà livrées
• Stats par client (nb de livraisons, taux d'échec, dernière visite)
• Favoris épinglés pour les clients critiques
• Couleurs custom par client pour repérage rapide
• Import / export CSV pour backup ou migration
• Export PDF annuaire imprimable

SCAN OCR DES BORDEREAUX
• Photo du bordereau → extraction automatique nom client, adresse, nb colis
• Géocodage cascade (BAN + Recherche-Entreprises + Photon)
• Format MESEXP (Messagerie Express) supporté
• Score de confiance + carte de validation si extraction incertaine

VALIDATION TERRAIN
• Bouton "Marquer livré" / "Marquer échec" + raison (absent, refusé...)
• Capture GPS au moment de la validation (preuve de passage)
• Notes par arrêt avec dictée vocale (clavier Android)
• Long press = copie l'adresse au presse-papier
• Mode "Tout marquer livré" pour les batches

VUE CARTE
• Carte OSM avec pin par arrêt coloré selon statut
• Trace de l'itinéraire optimisé en polyline
• Pin "moi" live en mode tournée en cours
• Mode plein écran pour la conduite
• Cache des tuiles pour réduire la conso data

STATISTIQUES
• Cumul 7 / 30 / 365 jours : tournées, arrêts, colis, distance, taux de réussite
• Bar chart 14 derniers jours
• Audit log par arrêt (qui a changé quoi, quand)

DIVERS
• Mode sombre pour la conduite de nuit
• Notifications locales (rappel tournée du lendemain)
• Bouton pause sur le chrono de tournée
• Templates de tournées récurrentes
• Multi-tournées par jour (matin / après-midi)
• Export PDF de tournée à partager par WhatsApp / mail

== CONFIDENTIALITÉ ==

100% local. Aucune donnée n'est envoyée à un serveur opti_route — il n'y a pas
de serveur. Les seules APIs externes interrogées sont publiques et gratuites
(BAN data.gouv, Recherche-Entreprises, OpenRouteService). Pas de tracking,
pas de pub, pas d'identifiant publicitaire.

== GRATUIT, SANS CB ==

L'app est entièrement gratuite. Les services tiers utilisés ont tous un plan
gratuit suffisant pour un usage individuel quotidien (jusqu'à 500 optimisations
de tournée par jour avec OpenRouteService). Tu n'auras jamais à sortir ta CB.
```
