Cas d'usage

Pour qui ?

Livreur independant

30 a 80 arrets/jour, gere son propre carnet et ses tournees. Phase 1 suffit largement, 0 cout.

Petit chef d'equipe (2-5)

Repartit les tournees, recoit les retours via WhatsApp/SMS. Phase 1 OK avec affectation + partage cible.

Entreprise structuree (5-20)

Plusieurs vehicules, planification quotidienne, facturation client. Phase 1 possible avec discipline ; Phase 2 recommande pour la sync.

Flotte pro (20+)

ERP integration, suivi RH, conformite RGPD avancee. Cible Phase 3 avec API publique + multi-vehicules VROOM.

Interesse par la version pro ?

La Phase 2 (sync multi-app temps reel) demarrera des qu'un budget ~25-50 USD/mois est valide. En attendant, la Phase 1 est dispo gratuitement.
